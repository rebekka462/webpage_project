Storage Units & Encoding

Understanding how data is stored and interpreted

Common Storage Units

  • Bit (b): Smallest unit of data in computing (0 or 1).
  • Byte (B): 8 bits, typically represents one character.
  • Kilobyte (KB): 1,024 Bytes.
  • Megabyte (MB): 1,024 KB.
  • Gigabyte (GB): 1,024 MB.
  • Terabyte (TB): 1,024 GB.

Data Encoding

Encoding determines how data is represented and interpreted.

  • ASCII: Uses 7 bits to encode characters (e.g., A = 65).
  • UTF-8: Variable-length encoding supporting all Unicode characters.
  • Base64: Encodes binary data as ASCII characters for transmission over text-based media.
  • Hexadecimal: Represents binary data in base-16 (e.g., 0x1F).

Why It Matters

Understanding storage units and encoding helps in:

  • Optimizing file storage and data transmission.
  • Ensuring compatibility across platforms and languages.
  • Debugging data representation issues.
