> HTML

HTML (HyperText Markup Language) is the standard language used to create web pages. It provides the structure and content of a webpage, using tags to define elements like:

  • Headings
  • Paragraphs
  • Images
  • Links
  • Forms
  • Tables

>About HTML

HTML (HyperText Markup Language) is the standard language used to create webpages. It allows developers to structure content such as text, images, links, and multimedia. HTML serves as the foundation of web development, working hand-in-hand with CSS for design and JavaScript for interactivity.

Every webpage you visit is built using HTML. Whether you're building a personal blog, a portfolio, or a full web application, HTML is the first step in bringing content to life on the internet.

    Common HTML Elements

    • Headings: <h1>, <h2>, <h3>
    • Paragraph: <p>This is a paragraph</p>
    • Link: <a href="https://example.com">Visit Site</a>
    • Image: <img src="image.jpg" alt="Description">
    • List: <ul><li>Item</li></ul>

    HTML Attributes

    Attributes provide additional information about an element. They're written inside the opening tag.

    Example:

    <img src="image.jpg" alt="A sample image">
    • src – Specifies the image source
    • alt – Describes the image if it can’t load
    • href – Used in <a> to define link URL

    Basic Text Formatting

    • <b>Bold Text</b> or <strong>
    • <i>Italic Text</i> or <em>
    • <u>Underlined Text</u>

    >Key Features of HTML

    • Uses tags to define elements
    • Tags can have attributes for extra info
    • Content is structured hierarchically
    • Works with CSS & JavaScript for full functionality
